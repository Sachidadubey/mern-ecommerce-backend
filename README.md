# MERN E-Commerce Backend

##  Overview
Professional production-ready E-Commerce backend built with Node.js, Express and MongoDB. 
Implements authentication, product management, cart, orders, payment and admin features.

##  Tech Stack
- Node.js
- Express
- MongoDB + Mongoose
- JWT Authentication
- bcrypt
- Nodemon
- ESLint

##  Project Structure
src/
  config/
  controllers/
  models/
  routes/
  middlewares/
  utils/
  services/

##  Setup Instructions
1️⃣ Clone repo
2️⃣ Install dependencies
3️⃣ Add .env file
4️⃣ Run project

##  Environment Variables
MONGO_URI=
JWT_SECRET=
PORT=

##  Run Commands
npm run dev
npm start

##  Features
- User Registration & Login
- JWT Authentication
- Admin Role Support
- Product CRUD
- Cart APIs
- Orders
- Payments (Later)
- Secure & Scalable Architecture
